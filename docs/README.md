# Documentation

This directory contains documentation for the BLE-Sensor project.

## Contents

- Hardware specifications
- Assembly instructions
- User guide
- API documentation
- Testing procedures

## Documents

### Hardware Documentation
- **Schematic Review**: Circuit design details
- **PCB Layout**: Board layout and layer stack
- **BOM**: Bill of materials with part numbers
- **Assembly Guide**: Step-by-step assembly instructions

### Firmware Documentation
- **API Reference**: Function and module documentation
- **BLE Protocol**: BLE service and characteristic definitions
- **Configuration Guide**: How to configure firmware parameters
- **Flashing Guide**: Instructions for programming the device

### User Documentation
- **Quick Start Guide**: Getting started with the sensor
- **User Manual**: Complete usage instructions
- **Troubleshooting**: Common issues and solutions

## Images

The `images/` subdirectory contains:
- Photos of the assembled board
- PCB renders
- Block diagrams
- Connection diagrams
- Screenshots

## Contributing Documentation

When adding documentation:
1. Use Markdown format for text documents
2. Use clear, descriptive filenames
3. Include images in the `images/` subdirectory
4. Keep documents concise and well-organized
5. Update this README when adding new documents
